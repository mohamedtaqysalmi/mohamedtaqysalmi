Started with "why does this code break" — still asking the same question, just at a higher level now.

Two diplomas: Artificial Intelligence (CMC Rabat-Salé-Kénitra) and Software Development (ISTA Mohamed El Fassi).
Mostly I learn by building things and breaking them until they work.

Right now that means chatbots with management dashboards, backend APIs designed to be handed off,
and automation pipelines that run without anyone watching them. Based in Morocco.
Writing Python more than speaking in meetings.

---

**What I actually deliver**

- Chatbots with full admin panels — built for production, not just to be demoed once
- REST APIs with FastAPI — clean enough to hand off, documented enough to maintain
- CI/CD pipelines — so deployments stop being a manual process that breaks on Fridays
- NLP systems in Arabic and French — real use cases pulled from real problems
